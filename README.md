# SpringBoot + Vue2仓库管理系统

## 功能模块

+ 登录
+ 管理员管理
+ 用户管理
+ 仓库管理
+ 物品分类管理
+ 物品管理
+ 记录管理
+ 出入库管理

## 如何启动

### 前端

```shell
npm install
npm run serve
```

### 后端

`SpringBoot2.7.10`
